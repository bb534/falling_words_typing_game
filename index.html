How to Play

Welcome to the Falling Words Typing Game! Here's how to play:

  1. Click the Start Game button to begin.
  2. Words will start falling from the top of the screen.
  3. Type the word in the input box and press Enter to score points.
  4. If a word reaches the bottom of the screen, the game is over.
  5. Try to score as many points as you can before the timer runs out!
